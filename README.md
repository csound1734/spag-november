## Getting started

* Download and install Csound
* Clone this repo (git clone) and navigate into it (cd spag-november)

then...

With vim installed:
* vim -S Nov2
* !csound test.csd -odac
* CTRL-C whenever you want to stop.

Without vim:
* csound test.csd -odac
* CTRL-C whenever you want to stop.
